# Signature-Base
signature-base is a submodule for my scanner tools LOKI and SPARK

# Directory Structure

- iocs - Simple IOC files (CSV)
- yara - YARA rules
- threatintel - Threat Intel API Receiver (MISP, OTX)
- misc - Other input files (not IOCs or signatures)
